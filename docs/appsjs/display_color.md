title: Display colors
---

Here is a table with the colors supported by the ST7789 display:

| Color   |  Hex   | Int value |
|:-------:|:------:| :----: |
|  Black  | 0x0000 |   0    |
|  White  | 0xFFFF | 65535  |
|   Red   | 0xF800 | 63488  |
|  Green  | 0x07E0 | 2016   |
|  Blue   | 0x001F |   31   |
|  Cyan   | 0x07FF |  2047  |
| Magenta | 0xF81F | 63519  |
| Yellow  | 0xFFE0 | 65504  |
| Orange  | 0xFC00 | 64512  |

**The int value is the value that JS API needs since JS doesn't support hex.**
